# Hybrid AI Assistant

A hybrid AI assistant using LangGraph, LangChain, and Ollama/OpenAI.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure environment:
   ```bash
   cp .env.example .env
   # Edit .env with your keys
   ```

3. Run the assistant:
   ```bash
   python hybrid_ai_assistant/main.py "Your query here"
   ```
